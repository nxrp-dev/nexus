# Nexus

Nexus is a family of Pascal projects focused on practical, readable, modern software development.

The documentation is organized around the current Nexus projects:

- **Nexus Pascal** — VS Code tooling for Free Pascal and Lazarus development.
- **Nexus UI** — cross-platform Pascal UI framework work.
- **Nexus Schema** — schema-driven generation and database tooling.
