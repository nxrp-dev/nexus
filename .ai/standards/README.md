# Standards

This folder contains stable project standards.

Standards describe how Nexus code, documentation, scripts, and other artifacts should be shaped. They are not task plans and should not collect one-off decisions from individual work sessions.

## Use

Folder-level `AGENTS.md` files should explicitly reference the standards that apply to that folder.

Do not ask Codex to infer which standards apply by scanning this whole folder. If a standard matters for a folder, name it from that folder's `AGENTS.md`.

## Current Standards

- `pascal.md` - Object Pascal / Free Pascal language, naming, unit, class, and object-model rules.
